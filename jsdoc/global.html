backgroundBox

This cell is used as a white background which is always behind all cells (nodes, bacground, links...) so that it can emulate the existence of the paper. That way the dimensions of the paper can be perceived by the contrast of this box with the greyish background. Events such as blank:pointerdown are no longer fired so they are emulated artificially when the user is clicking in the background box.
